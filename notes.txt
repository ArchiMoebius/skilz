The how:

 Get authors for repo

  Just authors
    git log --pretty=format:"%an%x09" | sort | uniq

  Authors and number of commits
    git shortlog --summary --numbered

 For a file, get the current number of lines per author.

    git blame --line-porcelain src/utils.js  |            sed -n 's/^author //p' |            sort | uniq -c | sort -rn

 Get days of work on project for an author:

    git log --author=`git config user.email` | grep "^Date" | awk '{print $2 " "  $3 " " $4}' | uniq | wc -l
